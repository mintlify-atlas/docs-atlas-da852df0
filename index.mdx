## How It Works

Nitrox uses Harmony patching to inject multiplayer functionality directly into Subnautica without modifying the game files.

<Steps>
  <Step title="Download the Launcher">
    Get the latest Nitrox Unlocked release from GitHub and extract it to your preferred location.
  </Step>
  <Step title="Configure Game Path">
    Point the launcher to your Subnautica installation directory.
  </Step>
  <Step title="Launch Multiplayer">
    Start the game through Nitrox and join or host a multiplayer session.
  </Step>
</Steps>

<Note>
  This is a modified version of the official Nitrox mod that works with non-Steam versions of Subnautica. It is not affiliated with or supported by the official Nitrox project.
</Note>
