# Welcome to Nitrox Unlocked

Nitrox Unlocked is a modified version of the [Nitrox](https://github.com/SubnauticaNitrox/Nitrox) multiplayer mod for [Subnautica](https://unknownworlds.com/subnautica/). This version enables multiplayer functionality for **non-Steam versions** of the game, making it accessible to players regardless of where they obtained their copy.

</Note>

## What is Nitrox Unlocked?

Nitrox Unlocked transforms Subnautica from a single-player survival experience into a cooperative multiplayer adventure. Explore the alien ocean world of 4546B with your friends, build bases together, and survive the depths as a team.

### Multiplayer Capabilities

- **Player Movement & Actions**: See other players swimming, using vehicles, and interacting with the world in real-time
- **Base Building**: Collaborate on base construction with synchronized building placement and deconstruction
- **Vehicle Synchronization**: Share submarines and other vehicles with proper docking/undocking coordination
- **Inventory Management**: Trade items and resources between players
- **Story Progression**: Complete story goals and radio messages as a group
- **PvP Support**: Optional player-versus-player combat (configurable in server settings)

### Server Features

The embedded server provides extensive configuration options:

- **Configurable Game Modes**: Survival, Freedom, Creative, and Hardcore modes
- **Auto-Save & Backup**: Automatic world saves with configurable intervals (default: every 2 minutes)
- **Player Management**: Set max connections (up to 100 players), passwords, and admin controls
- **World Customization**: Custom seeds, spawn positions, and player stats
- **LAN Discovery**: Automatic server discovery on local networks
- **UPnP Port Forwarding**: Automatic port forwarding for easier internet hosting

## Difference from Official Nitrox

The primary difference between Nitrox Unlocked and official Nitrox is **game version compatibility**:

- **Official Nitrox**: Requires a legitimate Steam version of Subnautica
- **Nitrox Unlocked**: Works with non-Steam versions of the game

Both versions share the same core multiplayer functionality, codebase, and features. Nitrox Unlocked maintains compatibility with the official Nitrox architecture while removing Steam verification checks.

## System Requirements

- **OS**: Windows 10/11, Linux (via Wine/Proton), or macOS
- **Game**: Subnautica (minimum version check enforced by launcher)
- **.NET Runtime**: .NET 7.0 or higher (included with launcher)
- **Network**: Port 11000 (default) must be accessible for multiplayer
